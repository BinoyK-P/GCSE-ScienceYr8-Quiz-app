This is an app for Science (GCSE Year-8) MCQ (Multiple Choice Quiz). The app is built using Streamlit platform and is in trial stage. The sidebar on the page displays two links - one to start the quiz and the oher to open the user comments page. On clicking the start quiz link, the quiz page opens. Once the user selects a Chapter from the selectbox, questions for the chapter are displayed (one at a time) with four options for answers. Choosing an option and clicking the 'Submit' button will lock the answer and display whether the answer is correct or wrong. User can then click the 'Next Question' button to get the next question from the chapter. User's score and chapter progress are shown at the bottom of the page. The user can select a new chapter at any time which will update the list of questions accordingly.
